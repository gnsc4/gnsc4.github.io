Loss Selling Cog - User Guide

This guide explains how to get started and use the loss selling features of the bot.

1. Verification

Before using most bot features, including loss selling, you need to be verified. This links your Discord account to your Torn account.

Verification might happen automatically when you join the server if you've linked your Discord account on Torn. If not, or if you need to reverify, follow these steps:

  1. Go to the official Torn Discord server invite link: torn.com/discord
  2. Join the official Torn Discord server if you haven't already.
  3. In any channel on the official Torn Discord server, type the command: !verify
  4. Follow the instructions provided by the Torn bot there to link your account.
  5. Return to the Discord server where you want to use this bot.
  6. Type the command: /verify

The bot should confirm your verification and update your nickname and roles if configured.

2. Add Your API Key

To participate in loss selling (and allow the bot to verify your attacks), you need to provide your Torn API key. This key needs **Limited Access** or greater permissions.

  1. Go to your Torn API Key management page: Torn API Preferences
  2. Create a new key if you don't have one suitable. Ensure it has at least "Limited Access".
  3. Copy the 16-character alphanumeric key.
  4. In this Discord server, use the command: /key add
  5. A pop-up window (modal) will appear. Paste your API key into the input field and submit.

The bot will confirm if the key was added successfully. Your key is stored securely (encrypted) and is only used for necessary checks like verifying your attacks for contracts.

You can remove your key anytime using /key remove.

3. Using Loss Selling Features

Once verified and your API key is added, you can interact with loss selling contracts:

  • List Available Contracts:

    To see which contracts are currently active and available for you to join, use the command:

    /contract list

  • Join a Contract:

    If you see a contract you want to participate in from the list, use its ID to join:

    /contract join contract_id:<ID> (Replace <ID> with the actual contract number)

    This will usually grant you access to a private channel for that specific contract.

  • Claim Losses:

    After performing losses against the target for a contract you've joined, claim them so they can be verified:

    /contract claim contract_id:<ID> losses_claimed:<Number> (Replace <ID> and <Number>)

    The bot will check if you have enough energy for the claimed losses based on your API key.

  • Check Your Balance/Status:

    To see your progress, verified losses, and payment balance across all contracts you've participated in within this server, use:

    /contract balance

  • Check Contract Status:

    To view the current details and participant list for a specific contract (even if you haven't joined), use:

    /contract status contract_id:<ID> (Replace <ID>)
